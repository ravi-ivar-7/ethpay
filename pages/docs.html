Getting Started

ethpay makes it easy to create and share BAT payment links. Here's how to get started:

  1. Connect your Brave Wallet
  2. Go to the Demo page
  3. Enter payment amount and recipient address
  4. Generate your payment link
  5. Share the link or QR code

Creating Payment Links

Payment links make it easy to request BAT payments. Simply provide:

  • Amount: How much BAT to request
  • Recipient Address: Your Ethereum address
  • Description: Optional payment description

Brave Wallet Integration

ethpay integrates seamlessly with Brave Wallet:

  • One-click wallet connection
  • Secure transaction signing
  • View your BAT balance
  • Automatic address detection
